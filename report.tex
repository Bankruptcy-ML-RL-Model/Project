\section{Project Overview}
This document details the end-to-end execution flow of the Corporate Bankruptcy Prediction System. The application is a full-stack machine learning pipeline designed to assess the financial health of Taiwanese companies using 48 distinct financial indicators sourced from historical economic data.

The platform goes beyond binary classification by providing three core functionalities:
\begin{enumerate}[label=\textbf{\arabic*.}]
    \item \textbf{Risk Scoring:} An XGBoost-driven probability engine calculating a localized 0--100\% bankruptcy risk.
    \item \textbf{Explainable AI (XAI):} SHAP (Shapley Additive Explanations) integration to extract the mathematically strongest positive and negative driving variables.
    \item \textbf{Strategy Optimization:} A Reinforcement Learning (PPO) simulator that dynamically tests millions of financial adjustments to generate an optimal 10-quarter recovery strategy.
\end{enumerate}

\section{Methodological Execution Flow}

\subsection{1. Predictive Inference (XGBoost Classifier)}
Because the underlying dataset exhibits a massive 9:1 imbalance (healthy vs bankrupt companies), the trained \texttt{XGBClassifier} was fitted using \texttt{scale\_pos\_weight} to heavily penalize False Negatives (predicting a bankrupt company is safe). The raw model probabilities were strictly adjusted using \textbf{Platt Scaling calibration} to ensure the output aligns with true geometric probabilities rather than purely ordinal ranks. 

The \texttt{risk\_classifier.py} module filters the output probability into one of 5 distinct UI classes based on calibrated thresholds (Safe: 0-20\%, Low Risk: 20-40\%, Moderate: 40-60\%, High Risk: 60-80\%, Critical: 80-100\%).

\subsection{2. Local Interpretability (SHAP XAI)}
To decouple the black-box nature of the Gradient Booster, a SHAP \texttt{TreeExplainer} maps the exact mathematical coefficients driving the current prediction. The Python backend extracts the Top 5 Risk Factors (Pushing risk higher, encoded red) and the Top 5 Protective Factors (Pulling risk lower, encoded blue), sorting them aggressively by magnitude. A dynamic parser translates the leading metrics into conversational English for executive review.

\subsection{3. Generative Strategy Simulation (Reinforcement Learning)}
If the company falls into a vulnerable tier ($\ge$ 5\% risk), the \texttt{rl\_api.py} spins up a Stable-Baselines3 PPO agent within a custom \texttt{Gymnasium} financial environment. Over a simulated 10-quarter timeline, the agent perturbs the most negatively skewed indicators identified by SHAP (max 5\% variance per step), hunting for the optimal mathematical path to drop the XGBoost probability to near-zero with the least aggressive real-world friction.

\section{Empirical Testing \& Model Results}

The core predictive engine achieved exceptional out-of-sample capability, heavily protected against overfitting through 5-Fold Stratified Cross-Validation.

\subsection{Threshold Optimization}
Because of the imbalanced 10\% default rate, the default 0.50 cutoff was fundamentally unsafe. Based on the \textbf{Youden's J Statistic} (Maximize: $TPR - FPR$), the optimal operational threshold was mathematically calculated at \textbf{$P = 0.0527$}. \\
\textit{At this cutoff: Sensitivity (TPR) is 95.5\%, with a False Positive Rate of only 6.11\%.}
